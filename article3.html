• Cryptography


Below is the specified requirements for the software.

  1. Transmitter can receive the vide file input to be sent to the receiver.
  2. Transmitter encrypts the data package, meanwhile, receiver decrypts the data package.
  3. The key that is used in the transmitter and receiver must be the same. If the key is different, then the receiver can't decrypt the data package which is resulted the video can't be displayed in the receiver.

In general, we can conclude that the modified video encryption algorithm will not affect the performance of the video streaming itself. The performance of the modified algorithm which is used to build the video encryption model is not affected by the length of the given key. The performance shall be constant, although the length of the given key can be differed from 1 to thousands of character(s). The application of the modified algorithm gives a lot of positive impacts such as the increasement of the security in sending data, faster computation, can be applied to a large size video, and it won't affect the performance of the video streaming itself.
